Waiting List

  • If a date is not available for a specific bungalow you can always add yourself to the waiting list.
  • If a cancellation has been placed, the first person in the waiting list will be sent an SMS notifying the cancellation.
  • You can add yourself to more than one waiting list.
  • Once you have added yourself to a waiting list check your messages daily.
  • Please contact the welfare manager of NSB/nearest NSB branch manager/second officer for more information.
